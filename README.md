# Лабораторная 7-8: Управление списком процессов

## Описание
Реализация связанного списка блоков управления процессами (PCB) на C++. Программа демонстрирует работу с динамическими структурами данных.

## 🏗️ Структура проекта
```
lab7-8/
├── CMakeLists.txt
├── README.md
├── .gitignore
└── src/
    ├── main.cpp
    ├── PCB.h
    ├── PCB.cpp
    ├── ListNode.h
    ├── ListNode.cpp
    ├── ProcessList.h
    └── ProcessList.cpp
```

## Сборка и запуск с CMake:
```bash
mkdir build
cd build
cmake ..
make
```

## Особенности
- Автоматическая сортировка при вставке
- Проверка на дубликаты
- Корректное освобождение памяти
- Обработка граничных случаев
